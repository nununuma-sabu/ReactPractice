# Signup Wizard (会員登録ウィザード) 🧙‍♂️

ReactとTypeScriptで実装された、シンプルな段階的（ステップ式）会員登録フォームです。
「入力」→「確認」→「完了」という画面遷移を、React Hooksを使って管理する基本的な実装パターンを示しています。

## 🚀 特徴

* **マルチステップ**: 画面遷移なしでスムーズにステップが進むUI。
* **型安全性**: TypeScriptを利用し、ステップの状態（`input`, `confirm`, `done`）を厳密に管理。
* **バリデーション**: 名前が未入力の状態では「次へ」に進めない制御。
* **シンプル設計**: 外部ライブラリを使わず、標準の `useState` のみでロジックを構成。

## 🛠 使用技術

* React
* TypeScript
* CSS (インラインスタイル)
